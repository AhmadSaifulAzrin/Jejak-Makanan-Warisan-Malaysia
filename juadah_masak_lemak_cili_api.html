Masak Lemak Cili Api

Tahukah Anda?

Masak lemak cili api ialah hidangan tradisional Melayu pedas yang menggabungkan ayam atau daging dengan santan, cili padi, kunyit hidup, dan rempah ratus lain. Bahan utama ialah cili padi dan kunyit hidup yang ditumbuk, kemudian ditumis dengan serai dan bahan lain. Santan dimasukkan secara perlahan untuk mengelakkan pecah minyak dan dimasak hingga semua bahan empuk.

Harga purata: RM 9.00.
