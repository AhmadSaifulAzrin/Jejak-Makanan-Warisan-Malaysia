Matlamat Pembangunan Aplikasi

Aplikasi mudah alih web statik ini dibangunkan sebagai Tugasan 1, Mobile Application Development Using HTML, CSS, and Bootstrap framework bagi kursus STIWK2114 (B) MOBILE PROGRAMMING. Fokus utama adalah untuk mendemonstrasikan keupayaan membina UI/UX yang responsif tanpa menggunakan *framework* JavaScript, sebaliknya bergantung sepenuhnya pada HTML dan CSS (termasuk Bootstrap pada skrin lain).



Tema berkonsepkan kepelbagaian budaya Malaysia yang dipilih untuk pembangunan web ini adalah berfokuskan kepada makanan tradisional Malaysia. Keunikan hidangan bagi setiap negeri dalam Malaysia menjadi pilihan utama kepada konsep tema web applikasi ini. Kepada pengguna (user), selain dihidangkan dengan info ringkas bagi setiap hidangan, mereka juga dibekalkan dengan maklumat multimedia dalam bentuk video dan peta lokasi restoran terkenal yg menyediakan sajian tersebut.
